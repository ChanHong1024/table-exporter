# table-exporter

A Chrome Extension that scans the current webpage for HTML tables and allows you to download them seamlessly as CSV or Excel files.

## Features

- **Automatic Detection:** Quickly scans and identifies HTML tables on the active webpage.
- **Multiple Formats:** Export data directly to **CSV** or **Excel** (.xlsx).
- **Easy to Use:** Simple, intuitive interface within the Chrome toolbar.

## Installation (Chrome Web Store)

*Note: The extension is currently pending review in the Chrome Web Store.*

Once approved, you can install **Table Exporter** directly from the store here:  
[Table Exporter on Chrome Web Store](https://chromewebstore.google.com/detail/gcehplmcfdgknnbjfghkneffilalibjp)

## Installation (Developer Mode)

1. Clone this repository or download the source code.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Toggle **Developer mode** on in the top right corner.
4. Click **Load unpacked** and select the `table-exporter` directory.

## Usage

1. Navigate to a webpage containing the table(s) you want to extract.
2. Click the **table-exporter** extension icon in your browser toolbar.
3. Select your desired export format to download the file.
